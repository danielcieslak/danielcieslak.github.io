# Personal Portfolio Website for Data Scientist Role

**Tailored for the Novo Nordisk Data Scientist position** (Commercial AI & Data team)

This is a modern, professional, single-page portfolio website built with Tailwind CSS. It is designed to help you stand out when applying for AI/ML and Data Science roles, especially in healthcare and regulated industries.

## What’s Included

- **Hero section** with strong value proposition aligned with healthcare AI impact
- **About Me** with bio and highlighted skills (Python, PyTorch, TensorFlow, MLOps, etc.)
- **Projects & Web Apps** – 3 detailed example projects (replace with yours). Includes live demo buttons that open rich modals with impact metrics, challenges, and learnings.
- **Articles & Insights** – Showcase thought leadership
- **Currently Learning** – Demonstrates growth mindset and alignment with MLOps + Responsible AI
- **CV / Resume section** – Clean preview + prominent download button
- Fully responsive, beautiful dark theme, smooth interactions, mobile menu
- Project detail modals with keyboard support (Esc to close)

## How to Customize (Quick Start)

1. **Open `index.html`** in any code editor (VS Code recommended).

2. **Replace all placeholders** (search for `[` to find them):
   - `[Your Full Name]`
   - `[Your Name]` (short version for logo)
   - `[YN]` (initials in logo)
   - `[Your City, Country]`
   - Education details, company names, thesis topic, etc.
   - Email address and Calendly link in contact section
   - GitHub / LinkedIn URLs

3. **Update the photo**:
   - Replace the `https://picsum.photos/id/1005/600/600` URL with your professional headshot (or upload to same folder and use relative path).

4. **Replace Projects** (most important for this role):
   - Edit the three project cards in the HTML.
   - Update titles, descriptions, tech stacks, impact numbers.
   - For the **Live Demo** buttons: link to your real deployed web apps (Streamlit, Gradio, or full web apps you build).
   - The modals are powered by JavaScript — you can easily add more projects by extending the `projectsData` array in the `<script>` section.

5. **Articles**:
   - Replace the example article cards with links to your real Medium / LinkedIn / personal blog posts.
   - If you don’t have articles yet, write 2–3 short ones on topics like:
     - MLOps pipelines
     - Deploying ML in healthcare / regulated environments
     - Lessons from clinical NLP or time-series forecasting

6. **CV Download**:
   - Replace `cv.pdf` with your actual resume PDF (put it in the same folder).
   - The preview section gives a nice summary — keep it concise.

7. **Contact**:
   - Update email and add a Calendly / scheduling link if you have one.

## Deployment (Free & Easy)

### Option 1: GitHub Pages (Recommended)
1. Create a new GitHub repo called `yourusername.github.io`
2. Upload `index.html` + any images/PDFs
3. Go to repo → Settings → Pages → Deploy from branch (main)
4. Your site will be live at `https://yourusername.github.io`

### Option 2: Netlify (Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the entire `personal-portfolio` folder
3. Get a beautiful URL instantly + custom domain later

### Option 3: Vercel
Same drag-and-drop experience, excellent performance.
